# Kart

A Mario Kart style racing game built with Python and Pygame.

## Prerequisites

- **Python 3.7+** - Make sure you have Python installed on your system
- **Pygame** - Game development library for Python

## Installation

1. **Clone or download this repository**

2. **Install Pygame**:
   ```bash
   pip install pygame
   ```

## Run the game

```bash
python main.py
```

Or on some systems:

```bash
python3 main.py
```
